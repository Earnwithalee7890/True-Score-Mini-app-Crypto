# TrueScore Smart Contracts

This directory contains the smart contracts used in the TrueScore Mini App.

## Deployed Contracts

### CheckIn.sol - Daily Check-In Contract

**Network:** Base Mainnet (Chain ID: 8453)  
**Contract Address:** `0xBD3aDb162D1C5c211075C75DFe3dCD14b549433A`  
**Deployed By:** earnwithalee@gmail.com  
**Verified:** Yes on BaseScan

#### Contract Features

- **Daily Check-Ins:** Users can check in once per day
- **Streak Tracking:** Consecutive check-ins increase user streaks
- **Reputation System:** Build reputation score with consistent engagement
- **Anti-Spam Protection:** 0.000001 ETH fee prevents spam
- **Stats Tracking:** Total check-ins, users, and contract balance

#### Functions

##### Public Functions

- `checkIn()` - Perform daily check-in (payable: 0.000001 ETH)
- `getUserStats(address)` - Get user's check-in statistics
- `getContractStats()` - Get overall contract statistics
- `getCurrentDay()` - Get current day index

##### Owner Functions

- `withdraw()` - Withdraw collected fees (owner only)
- `transferOwnership(address)` - Transfer contract ownership

#### Usage Example

```javascript
// Using wagmi/viem
import { useWriteContract } from 'wagmi'

const { writeContractAsync } = useWriteContract()

await writeContractAsync({
  address: '0xBD3aDb162D1C5c211075C75DFe3dCD14b549433A',
  abi: CheckInABI,
  functionName: 'checkIn',
  value: BigInt(1_000_000_000_000), // 0.000001 ETH
  chainId: 8453 // Base Mainnet
})
```
